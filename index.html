Mobile App (React Native & TypeScript)

This mobile app is developed using React Native. The idea for the app came from my own passion for fitness, especially strength training. I wanted to create a tool that would help users easily log and track their workouts, including details such as exercise type, duration, and progression over time. All workout data is stored securely on the device, ensuring privacy and easy access. The app is designed with a user-friendly interface to provide an intuitive experience, helping users stay motivated, track their progress, and stay organized in their fitness journey.

Web App (Java, Spring, PostgreSQL, React, JavaScript)

This web application is developed using Java, Spring, PostgreSQL, React, and JavaScript. The app enables users to browse various topics, each containing detailed notes and information on specific subjects. Users can easily select a topic and read through study material or notes associated with it. The application features an admin-only login system, where only authorized users can log in to add new courses, topics, and relevant content. This ensures that the platform remains organized and up-to-date with accurate and relevant information. The idea for this project came from my own need to store and organize study notes under different categories. I wanted a simple yet effective way to access and review my notes at a later time, making studying more efficient.

Technologies Used: Java, Spring, PostgreSQL, React, JavaScript

Check out the project on GitHub.
